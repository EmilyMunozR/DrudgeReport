============================================
FUNCIONAMIENTO DEL MODO OSCURO:
============================================
Cuando se hace clic en el switch de tema:
1. Se añade la clase "dark-mode" al <body>
2. Las variables CSS cambian sus valores (!important)
3. Todos los elementos con estilos dark-mode se activan
4. La perilla del switch se mueve 30px a la derecha
5. El fondo cambia de blanco (#f4f4f4) a negro (#111111)
6. Los textos cambian de negro a blanco (#f4f4f4)

============================================
FUNCIONAMIENTO DEL SWITCH DE IDIOMA:
============================================
Cuando se hace clic en el switch de idioma:
1. Se alterna entre ES y EN
2. Se añade/remueve la clase "en-mode" al .lang-switch
3. La perilla se mueve 40px a la derecha (EN) o izquierda (ES)
4. Se carga el archivo JSON correspondiente (lang/es.json o lang/en.json)
5. Se actualizan todos los elementos con data-translate
